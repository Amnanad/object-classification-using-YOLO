# object-classification-using-YOLO
This project uses the YOLO object detection model to identify and classify different types of flowers from a custom dataset. It includes code for training, testing, and inference. The dataset is provided via external links due to GitHub file size limits. https://drive.google.com/drive/folders/161vvOs6QanWdSj42hB8Qbjqi5sDq5E1C?usp=sharing
